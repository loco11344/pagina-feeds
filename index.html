Acerca del Programa

Conoce nuestro programa de Bachillerato Técnico en Informática, diseñado para formar profesionales competentes en el área tecnológica.

Formación Integral en Tecnología

Nuestro programa de Bachillerato Técnico en Informática combina una sólida formación académica con habilidades técnicas especializadas en el área de la informática y las tecnologías de la información.

Los estudiantes adquieren conocimientos en programación, redes, bases de datos, desarrollo web y mantenimiento de equipos, preparándolos para continuar estudios superiores o incorporarse al mercado laboral.

Contamos con laboratorios equipados con tecnología de punta y docentes altamente calificados que guían a los estudiantes en su proceso de aprendizaje.

Módulos Técnicos Especializados

Conoce nuestras áreas de especialización técnica que preparan a los estudiantes para el mundo laboral actual

Mantenimiento de Hardware

Diagnóstico, reparación y mantenimiento preventivo y correctivo de equipos de cómputo.

  • Ensamblaje de computadoras
  • Diagnóstico de fallas
  • Reemplazo de componentes
  • Limpieza y mantenimiento

Soporte de Software

Instalación, configuración y solución de problemas en sistemas operativos y aplicaciones.

  • Instalación de sistemas operativos
  • Configuración de controladores
  • Solución de errores comunes
  • Optimización de rendimiento

Atención al Usuario

Habilidades de comunicación y resolución de problemas técnicos para soporte a usuarios finales.

  • Técnicas de atención al cliente
  • Documentación de problemas
  • Guías de solución paso a paso
  • Soporte remoto y presencial

Diseño Web

Principios de diseño, maquetación y creación de interfaces web atractivas y funcionales.

  • Diseño UI/UX
  • HTML5 y CSS3 avanzado
  • Diseño responsive
  • Frameworks CSS (Bootstrap)

Desarrollo Frontend

Programación del lado del cliente para crear interfaces web interactivas y dinámicas.

  • JavaScript ES6+
  • Manipulación del DOM
  • APIs y AJAX
  • Frameworks modernos (React)

Desarrollo Backend

Creación de servidores, APIs y lógica de negocio para aplicaciones web completas.

  • Node.js y Express
  • Bases de datos para web
  • APIs RESTful
  • Autenticación y seguridad

Bases de Datos

Diseño, implementación y administración de sistemas de bases de datos relacionales.

  • Modelado ER
  • SQL avanzado
  • MySQL / PostgreSQL
  • Normalización de bases de datos

Programación Estructurada

Fundamentos de programación usando lenguajes estructurados como C y Python.

  • Algoritmos y pseudocódigo
  • Estructuras de control
  • Funciones y procedimientos
  • Manejo de archivos

Programación Orientada a Objetos

Desarrollo de software usando paradigma orientado a objetos con Java o C#.

  • Clases y objetos
  • Herencia y polimorfismo
  • Encapsulamiento y abstracción
  • Patrones de diseño básicos

Sistemas Operativos

Administración y configuración de sistemas operativos Windows y Linux.

  • Instalación y configuración
  • Administración de usuarios
  • Scripting en PowerShell/Bash
  • Virtualización (VirtualBox, VMware)

Redes de Computadoras

Fundamentos de redes, configuración de equipos de red y protocolos de comunicación.

  • Modelo OSI y TCP/IP
  • Configuración de routers y switches
  • Protocolos de red
  • Redes inalámbricas

Seguridad Informática

Protección de sistemas, redes y datos contra amenazas y vulnerabilidades.

  • Firewalls y cortafuegos
  • Autenticación y cifrado
  • Detección de intrusiones
  • Políticas de seguridad

Procesamiento de Textos

Uso avanzado de procesadores de texto para creación de documentos profesionales.

  • Formatos y estilos avanzados
  • Índices automáticos
  • Combinación de correspondencia
  • Trabajo colaborativo

Hojas de Cálculo

Análisis de datos, fórmulas complejas y automatización con hojas de cálculo.

  • Fórmulas y funciones avanzadas
  • Macros y VBA básico
  • Tablas dinámicas y gráficos
  • Análisis de datos empresariales

Presentaciones Digitales

Creación de presentaciones profesionales y efectivas para diversos contextos.

  • Diseño de diapositivas
  • Animaciones y transiciones
  • Integración multimedia
  • Presentaciones interactivas

Ofimática en la Nube

Trabajo colaborativo usando suites ofimáticas en línea como Google Workspace y Office 365.

  • Trabajo colaborativo en tiempo real
  • Almacenamiento en la nube
  • Compartición y permisos
  • Automatización con herramientas en línea
